The AI tool created for the final project is designed to help visualize the early stages of an illustration project by generating visual elements such as color palettes. By entering a single keyword or short phrase, such as “ocean” or “1970s Tokyo,” the user prompts the tool to generate a palette of five colors accompanied by a short written description. This description explains how and why each color connects to the input, drawing on associations like atmosphere, lighting, or emotional tone. In this way, the tool does not just produce colors randomly, but attempts to contextualize them, giving the user a starting point that feels intentional rather than arbitrary. At the same time, the tool offers flexibility in how much control the user wants. Users can modify aspects such as temperature, brightness, saturation, and chromacity to refine the results toward a specific visual direction. Ultimately, the goal of this project is to create a tool that assists rather than replaces the artist. Many existing AI systems tend to generate fully realized images, which can sometimes overshadow the artist’s own decision-making process. Therefore, this tool is intentionally limited in scope. It aims to help artists begin, offering a visual and conceptual entry point, while leaving the majority of the creative work, interpretation, and execution in the hands of the user.
